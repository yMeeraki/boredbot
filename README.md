# BoredBot

A simple JavaScript project that suggests a random activity using the Bored API.

## Features

* Fetches a random activity from an API
* Updates the page dynamically
* Built with HTML, CSS, and JavaScript

## Technologies

* HTML
* CSS
* JavaScript
* Fetch API

## What I Learned

* Client and Server
* Request–Response Cycle
* APIs
* JSON
* Fetch API
* DOM Manipulation

## Getting Started

Clone the repository and open `index.html` in your browser, or run it using Live Server.
